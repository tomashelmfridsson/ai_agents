The broader project evaluates multiple LLMs, local versus cloud inference, and alternative agentic orchestration patterns for QA. The current public solution uses Hugging Face-backed model execution, while the orchestration, routing, memory handling, feedback limits, and observability are implemented in this app.

This means the app is not just a thin client for hosted agents. It keeps the project's own orchestration model, and the codebase still preserves the path for running the project's own in-app agents. A natural next UI step is a clear mode switch between in-app agents and Hugging Face-hosted execution.

Alternative solution QA Agent LangGraph Added as a comparison track for graph-based orchestration. It makes node transitions, states, and flow control more explicit, which helps evaluate what LangGraph adds compared with the app's own orchestrator-driven approach.
