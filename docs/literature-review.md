# 3. Litteraturstudie

Detta kapitel sammanfattar aktuell forskning och relevanta kunskapsspår inom:

- AI Agents
- Multi-Agent Systems
- Agentic Software Engineering
- AI-assisterad testning
- Software Quality Assurance
- Verification and Validation

Varje forskningsarbete bör analyseras utifrån:

- syfte
- metod
- resultat
- relevans för studiens syfte

Litteraturstudien ska alltså fokusera på vad tidigare forskning, artiklar och ramverksnära källor säger om området, inte på den egna lösningens arkitektur eller implementation. Beskrivningar av den egna prototypen, designbeslut, agentflöden, experimentupplägg och praktiska utvärderingar hör därför inte längre hemma i denna fil utan i POC-rapporten.

Det innebär också att följande innehåll nu bör betraktas som utbrutet från litteraturstudien:

- projektets forskningsfråga, delmål och avgränsning finns i Projektbrief
- centrala begrepp och kunskapslista finns i Teoretisk bakgrund och centrala begrepp
- agentramverksjämförelser kopplade till den egna lösningen finns i AI Agents POC Rapport
- systemarkitektur, designbeslut, implementation och experimentbeskrivning finns i AI Agents POC Rapport

Om det senare behövs en mer formell litteraturgenomgång kan detta kapitel byggas ut med separata underavsnitt per forskningsområde eller per källa.
